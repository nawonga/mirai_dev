# Claude Code Developer Guide for Mirai Space

## 1. 프로젝트 개요
이 워크스페이스는 라즈베리파이 5에서 구동되는 스마트홈 및 45cm 큐브 해수어항 제어 시스템인 **'미라이 스페이스(Mirai Space)'**의 통합 저장소이다. 

## 2. 핵심 아키텍처 규칙 (필독)
시스템의 모든 아키텍처 원칙, 각 모듈의 역할, 그리고 안전 규칙은 반드시 `OPERATING.md` (Mirai Space Constitution)를 읽고 엄격하게 준수해야 한다. 코드를 수정하거나 새로운 기능을 제안할 때 `OPERATING.md`의 원칙을 위배해서는 안 된다.

## 3. 환경 분리 원칙 (Dev vs Prod)
현재 프로젝트는 개발 환경(`dev`)과 운영 환경(`prod`)으로 완벽히 분리되어 있다.
- **경로 구조:** - `/home/nawonga/MiraiProject/dev/` : 모든 새로운 기능 개발, 테스트, 패키지 설치가 이루어지는 곳.
  - `/home/nawonga/MiraiProject/prod/` : 검증이 완료된 코드가 배포되어 실제 라즈베리파이에서 서비스되는 곳.
- **가상환경(venv) 분리:** 각 환경은 독립된 Python venv를 가진다. 패키지 설치나 스크립트 실행 시 자신이 현재 `dev`인지 `prod`인지 반드시 경로를 확인하라.
- **작업 원칙:** 클로드 코드는 오직 `dev` 환경에서만 코드를 작성하고 수정한다. 절대 `prod` 환경의 코드를 직접 수정하지 않는다.

## 4. 코딩 컨벤션 및 주의사항
- **Python:** 주 언어는 Python 3를 사용하며, API는 FastAPI, 데이터 검증은 Pydantic을 활용한다.
- **의존성:** 라즈베리파이 하드웨어 리소스를 고려하여 무거운 패키지 추가는 지양한다.
- **보안:** `.env` 파일에 있는 API 키(Gemini, MQTT 등)는 절대 코드에 하드코딩하거나 깃(Git) 커밋에 포함시키지 않는다.
